GTM Entegrasyon Kontrolü

Console'da Kontrol Edilecekler:

  1. Sitenizi açın: https://www.modabase.com.tr
  2. Sağ tık → İncele (Inspect)
  3. Console sekmesine geçin
  4. Aşağıdaki komutları yazın:

Komut 1: dataLayer Kontrolü

console.log('dataLayer var mı?', typeof dataLayer !== 'undefined');
console.log('dataLayer:', dataLayer);

Komut 2: GTM Yüklendi mi?

console.log('GTM yüklendi mi?', typeof google_tag_manager !== 'undefined');
console.log('GTM ID:', Object.keys(google_tag_manager || {}));

Komut 3: GTM Script Kontrolü

// GTM script tag'ini kontrol et
const gtmScript = document.querySelector('script[src*="googletagmanager.com/gtm.js"]');
console.log('GTM Script:', gtmScript ? gtmScript.src : 'Bulunamadı');

// GTM noscript iframe kontrolü
const gtmIframe = document.querySelector('iframe[src*="googletagmanager.com/ns.html"]');
console.log('GTM Iframe:', gtmIframe ? gtmIframe.src : 'Bulunamadı');

Beklenen Sonuçlar:

  • ✅ dataLayer var mı?: true
  • ✅ GTM yüklendi mi?: true
  • ✅ GTM ID: ['GTM-KLXBLCV6']
  • ✅ GTM Script: https://www.googletagmanager.com/gtm.js?id=GTM-KLXBLCV6
  • ✅ GTM Iframe: https://www.googletagmanager.com/ns.html?id=GTM-KLXBLCV6

Network Tab Kontrolü:

  1. İncele → Network sekmesi
  2. Sayfayı yenile (F5)
  3. Filter'a "gtm" yazın
  4. Şunları görmelisiniz:
    • gtm.js?id=GTM-KLXBLCV6 (Status: 200)
    • Eğer AdBlocker varsa kapatın

Google Search Console Doğrulama:

Görüntüdeki ekrandan "Google Analytics" yöntemini seçerseniz:

  1. GTM'de Google Analytics etiketi aktif olmalı
  2. GA4 property'si oluşturulmuş olmalı
  3. GA4 kodu GTM üzerinden yayınlanmış olmalı
